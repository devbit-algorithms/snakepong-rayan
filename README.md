
# Nodige componenten

```TEXT
De nodige componenten zijn de volgenden: 

-Een slang
-Een Bal 
-Een venster waarin het kan gespeeld worden  
-Een keur voor de bal(console, tekens vernaderen)
-Een manier om de slang te vergroten in functiie van het scoren.
-Een object die continu beweegst in functie van de bal
-De achtergrond wit ? 
-Een bal met dimensies
-Een reset mode als de slang een muur raakt
-Een vorm verandering als die scoort(laten blinken). 
-Een interface voor de beweging (slang positie) (keys)
-opties voor meerdere speelers 
-interface met gebruik van de cursor
-De inkom maken waarin je de speler verwelkom 
```
